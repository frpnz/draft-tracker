# Tournament modes

This tool supports multiple event formats.

## 1) 1v1 round-robin (single / Bo1)
- Every player plays every other player once.
- Each match is a single game.
- Ranking tie-breaks: wins -> head-to-head -> delta life -> random draw.

## 2) 1v1 round-robin (Bo3)
- Every player plays every other player.
- Each match is Best of 3 games.
- Ranking tie-breaks: match wins -> game wins -> head-to-head -> delta life -> random draw.

## 3) Multiplayer commander (3 to 12 players)
- Players are split into 1-3 tables.
- You enter placements with: place 1 = winner.
- If there are at least 2 main tables, you can create a final table.

## 4) Two groups -> Top 2 -> Semifinals -> Final (6 to 10 players)
- Participants are randomly split into two groups (A and B).
- Group stage: round-robin Bo1 inside each group.
- Top 2 from each group advance.
- Semifinals: A1 vs B2 and B1 vs A2.
- Final is played between semifinal winners.
- Playoff format (Bo1 or Bo3) is selectable when creating the event.
